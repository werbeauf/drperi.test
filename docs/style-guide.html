03 · Schnittgewichte

Wann welcher Weight?

Empfehlung: Lade nur 400, 500, 600, 700 — das deckt 95% der Anwendungsfälle ab und spart ~60% Ladezeit gegenüber dem vollen Set.

300
Light
Großes Display, niemals Body
Nur ab 32px+. Unter 24px wirkt Light auf Bildschirmen ausgewaschen.
400
Regular
Standardfließtext, Absätze, Lead-Texte
Das Arbeitstier. 90% des Body-Texts läuft hier.
500
Medium
Hervorhebungen, Buttons, Labels, Navigation
Ideal für UI-Elemente und mittlere Akzente. Subtile Betonung im Fließtext.
600
SemiBold
Subheadings, kleinere Headlines, Card-Titel
Standard für H3–H5. Klare Hierarchie ohne Wucht.
700
Bold
Hauptüberschriften H1, H2, starke CTAs
Reserviert für die wichtigsten visuellen Anker.
800–900
Extra/Black
Nur für Hero-Display & Logo
Sparsam einsetzen — sonst Werbeplakat-Effekt. Nie unter 36px verwenden.

06 · Best Practices

Do & Don't

Die wichtigsten Regeln für saubere Montserrat-Typografie.

✓ Do
Body in 400, max-width: 65ch, line-height ≥ 1.6, Akzent #495f76 nur für Links, CTAs und Eyebrows
✗ Don't
Body in 300 unter 18px, ALL CAPS in langen Sätzen, mehr als 3 Schnitte parallel, Akzent für Fließtext
✓ Do
clamp() für fluide Skalierung — keine harten Breakpoint-Sprünge, smoothe Anpassung an jeden Viewport
✗ Don't
Feste px-Werte mit @media-Queries stapeln — erzeugt sichtbare Sprünge bei Größenwechseln
✓ Do
Body-Text in #2f4053 (Akzent-Familie 700) — harmoniert besser als reines Schwarz
✗ Don't
Reines #000000 für Fließtext — wirkt zu hart, ermüdet die Augen bei langem Lesen
✓ Do
font-display: swap setzen, preconnect zu Google Fonts, woff2 verwenden
✗ Don't
Alle 9 Schnitte laden — verbraucht 600KB+ und verzögert First Paint
07 · Setup

CSS — Copy & Paste

Komplettes Setup mit fluider clamp()-Skala. DSGVO-konform via Self-Hosting empfohlen (siehe Hinweis unten).

/* 1. Google Fonts (HTML <head>) */
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&display=swap" rel="stylesheet">

/* 2. CSS Custom Properties — Fluid Type Scale */
:root {
  --font-base: 'Montserrat', system-ui, -apple-system, sans-serif;

  /* Akzentfarben */
  --color-accent:        #495f76;
  --color-accent-50:     #f0f3f7;
  --color-accent-100:    #d8dfe6;
  --color-accent-300:    #a3b1c2;
  --color-accent-700:    #2f4053;
  --color-accent-900:    #1a2533;
  --color-text:          #2f4053;
  --color-text-muted:    #5a6f85;
  --color-heading:       #1a2533;

  /* Fluid Type Scale (360px → 1280px Viewport) */
  --fs-display:  clamp(2.5rem,    1.5rem  + 4.35vw, 5rem);       /* 40 → 80px */
  --fs-h1:       clamp(2rem,      1.3rem  + 3.04vw, 3rem);       /* 32 → 48px */
  --fs-h2:       clamp(1.625rem,  1.13rem + 2.17vw, 2.25rem);    /* 26 → 36px */
  --fs-h3:       clamp(1.375rem,  1.05rem + 1.41vw, 1.75rem);    /* 22 → 28px */
  --fs-h4:       clamp(1.1875rem, 0.99rem + 0.87vw, 1.375rem);   /* 19 → 22px */
  --fs-h5:       clamp(1.125rem,  1rem    + 0.54vw, 1.25rem);    /* 18 → 20px */
  --fs-h6:       clamp(1rem,      0.92rem + 0.33vw, 1.125rem);   /* 16 → 18px */
  --fs-lead:     clamp(1.0625rem, 0.92rem + 0.65vw, 1.25rem);    /* 17 → 20px */
  --fs-body:     clamp(0.9375rem, 0.88rem + 0.27vw, 1rem);       /* 15 → 16px */
  --fs-small:    clamp(0.8125rem, 0.79rem + 0.22vw, 0.875rem);   /* 13 → 14px */
  --fs-caption:  clamp(0.6875rem, 0.66rem + 0.22vw, 0.75rem);    /* 11 → 12px */
  --fs-overline: clamp(0.75rem,   0.68rem + 0.33vw, 0.875rem);   /* 12 → 14px */

  /* Komponenten-Skala */
  --fs-card-title:    clamp(1.125rem,  0.97rem + 0.65vw, 1.3125rem);  /* 18 → 21px */
  --fs-product-title: clamp(1rem,      0.94rem + 0.27vw, 1.0625rem);  /* 16 → 17px */
  --fs-tile-title:    clamp(0.9375rem, 0.91rem + 0.13vw, 0.96875rem); /* 15 → 15.5px */
  --fs-price:         clamp(1.25rem,   1.1rem  + 0.65vw, 1.4375rem);  /* 20 → 23px */
  --fs-price-old:     clamp(0.875rem,  0.85rem + 0.11vw, 0.9375rem);  /* 14 → 15px */
  --fs-meta:          clamp(0.75rem,   0.73rem + 0.11vw, 0.8125rem);  /* 12 → 13px */
  --fs-tag:           clamp(0.6875rem, 0.66rem + 0.13vw, 0.75rem);    /* 11 → 12px */
}

/* 3. Base Styles */
html { font-size: 100%; }
body {
  font-family: var(--font-base);
  font-size: var(--fs-body);
  line-height: 1.75;
  color: var(--color-text);
  font-weight: 400;
  -webkit-font-smoothing: antialiased;
  text-rendering: optimizeLegibility;
}

/* 4. Headings — alle fluide */
h1 { font-size: var(--fs-h1); line-height: 1.15; font-weight: 700; letter-spacing: -0.02em;  color: var(--color-heading); }
h2 { font-size: var(--fs-h2); line-height: 1.2;  font-weight: 700; letter-spacing: -0.015em; color: var(--color-heading); }
h3 { font-size: var(--fs-h3); line-height: 1.3;  font-weight: 600; letter-spacing: -0.01em;  color: var(--color-heading); }
h4 { font-size: var(--fs-h4); line-height: 1.35; font-weight: 600; color: var(--color-accent-700); }
h5 { font-size: var(--fs-h5); line-height: 1.4;  font-weight: 600; color: var(--color-accent-700); }
h6 { font-size: var(--fs-h6); line-height: 1.45; font-weight: 600; color: var(--color-accent-700); }

/* 5. Text Elements */
p     { font-size: var(--fs-body); max-width: 65ch; margin-bottom: 1.25rem; }
a     { color: var(--color-accent); font-weight: 500; text-underline-offset: 3px; }
small { font-size: var(--fs-small); color: var(--color-text-muted); }
strong { font-weight: 600; }

/* 6. Utility Classes */
.display  { font-size: var(--fs-display); font-weight: 800; line-height: 1.1; letter-spacing: -0.025em; }
.lead     { font-size: var(--fs-lead); line-height: 1.6; font-weight: 400; }
.eyebrow  {
  font-size: var(--fs-overline); font-weight: 600;
  letter-spacing: 0.12em; text-transform: uppercase;
  color: var(--color-accent);
}
.caption  { font-size: var(--fs-caption); font-weight: 500; color: var(--color-text-muted); }

/* 7. Buttons */
.btn {
  display: inline-block;
  padding: 12px 24px;
  border-radius: 6px;
  font-size: var(--fs-small);
  font-weight: 600;
  font-family: var(--font-base);
  cursor: pointer;
  transition: all 0.15s ease;
}
.btn-primary   { background: var(--color-accent); color: white; border: none; }
.btn-primary:hover { background: var(--color-accent-700); }
.btn-secondary {
  background: transparent;
  color: var(--color-accent);
  border: 1.5px solid var(--color-accent);
}
.btn-secondary:hover { background: var(--color-accent-50); }

clamp() erklärt: Die Formel clamp(MIN, PREFERRED, MAX) nimmt den mittleren, viewport-basierten Wert — gedeckelt durch Min und Max. Beispiel --fs-h1: bei 360px Viewport = 32px, bei 1280px = 48px, dazwischen lineare Interpolation. Keine Media-Queries nötig.

DSGVO-Hinweis (Österreich): Google Fonts via CDN überträgt IP-Adressen an Google in den USA. Für volle DSGVO-Konformität die Schriften lokal hosten — z.B. via Fontsource oder als woff2-Files direkt vom Server ausliefern.
